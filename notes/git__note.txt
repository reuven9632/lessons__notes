git								- показывает все команды git и описывает их значение
ls 								- показывает содержимое папки в которой нахожусь
cd name_folder							- заходит в папку которую укажу
git clone https://github.com/reuven9632/lessons__notes.git	- копирует по ссылке url репозиторий из инт. на компьютер в папку в которой нахожусь	
git status							- показывает статус нашего проекта : файлы папки и загружены ли они 
								на git или они только локальные

git add name-of-file.html					- добавляю файл к коммиту
git add -A							- добавляю файл, и все обновления в других файлах, к коммиту
git commit -m "text-of-commit"					- добавляю комит (файл все еще на компьютере)
git push							- публикует коммит (в случае если это первый раз то запросит логин и пароль, 
								файл переноситься в инт.)
git pull							- обновление репозитория (пример: если второй разработчик написал 
								кусок кода то он обновиться или добавяться новые файлы)




clear								- очищает командную стоку (не отменяя предыдущих команд а просто расчищает 
								для удобного пользования чтобы не было ничего лишнего перед глазами)
